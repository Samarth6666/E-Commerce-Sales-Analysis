# E-Commerce Sales Analysis Dashboard

An interactive **End-to-End E-Commerce Sales Analysis Dashboard** developed using **Power BI**, **Python**, **Excel**, **DAX**, and **Figma**.

This project transforms raw sales data into meaningful business insights through interactive dashboards, helping stakeholders monitor sales performance, customer behavior, product performance, and geographical trends.



# Project Overview

The objective of this project is to provide a comprehensive business intelligence solution for an e-commerce business by analyzing sales data across multiple dimensions.

The dashboard enables users to:

- Monitor overall business performance
- Analyze sales trends
- Identify top-performing products
- Understand customer purchasing behavior
- Evaluate regional sales performance
- Make data-driven business decisions

# Dashboard Pages

## Executive Overview

Provides a high-level summary of overall business performance.

KPIs Included:

- Total Profit

Visuals:

- Top Products
- Sales by Region



## Sales Analysis

Focuses on sales trends and revenue performance.

KPIs:

- Average Order Value

Visuals:

## Product Performance

# Key Business Insights

- Sales remained consistent with noticeable seasonal peaks.
- A small number of products generated a significant share of revenue.
- Repeat customers contributed substantially to total sales.
- Customer spending varied across different cities and regions.
- Sales were concentrated in major cities, indicating opportunities for regional expansion.
